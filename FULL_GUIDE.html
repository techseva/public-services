Complete Easy Setup Guide (Start to Finish)

Follow these steps one by one. Do not skip any.

STEP 1: Open the Setup Tool

  1. Go to your folder where the files are.
  2. Double-click on the file named setup.html.
  3. It will open in Chrome or Edge. Keep this open.

STEP 2: Create a Google Firebase Project

  1. Click the "Open Firebase" button inside the setup tool (or go to console.firebase.google.com).
  2. Log in with your Gmail account.
  3. Click the big white rectangle that says "Create a project" (or "Add project").
  4. Enter a name: Type My-App.
  5. Click Continue.
  6. Google Analytics: Turn the switch OFF (Disable it).
  7. Click Create project.
  8. Wait a moment, then click Continue when it's ready.

STEP 3: Register Your Web App

  1. You will see the "Project Overview" page.
  2. Look for 3 circle icons (iOS, Android, Web).
  3. Click the Web Icon </> (It looks like brackets with a slash).
  4. App nickname: Type Website.
  5. Leave "Firebase Hosting" UNCHECKED (Do not check it).
  6. Click Register app.
  7. STOP! Do not close this page yet.

STEP 4: Copy the Secret Code

  1. On the screen, you will see a block of code code.
  2. Look for the part that starts with: const firebaseConfig = { ... };
  3. Select and Copy that whole block of text.
  4. (If you accidentally closed it, click the Gear Icon ⚙️ -> Project Settings -> Scroll down to "Your apps").

STEP 5: Turn ON the Database (Very Important!)

  1. On the left side menu of Firebase, click Build.
  2. Click Realtime Database.
  3. Click Create Database.
  4. Location: Leaves as default (US), click Next.
  5. Security Rules: Select "Start in Test Mode" (The second option).
  6. Click Enable.

STEP 6: Connect & Finish

  1. Go back to my setup.html tab (from Step 1).
  2. Paste the code you copied in Step 4 into the box.
  3. Click CONNECT NOW.
  4. If successful, it will say "Success" and open your app.

Done! Now any update you add will be seen by everyone using your app.
